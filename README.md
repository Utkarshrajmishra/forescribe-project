## 🛠️ Getting Started (Bun)

```bash
# Clone the repo
git clone https://github.com/your-username/your-repo.git
cd your-repo

# Install dependencies
bun install

# Run dev server
bun run dev
```

Open http://localhost:3000

##  Build

```bash
bun run build
bun run start
```
